#Vamos Juntas

Repositório do cliente da aplicação do Vamos Juntas


##Instalação

Necessário instalar:

* NPM
* Bower
* Karma cli
* PhantomJS
* Ionic

####NPM
`brew install node`

####Bower
`npm install -g bower`

####Karma cli
`npm install -g karma-cli`

####PhantomJS
`npm install -g panthomjs`

####Ionic
`npm install -g cordova ionic ios-sim`

#### How to run the tests

`karma start karma.conf.js`
